# Ajay Bandiwaddar — Portfolio

This repository hosts a compact portfolio site focused on AI and Reinforcement Learning work. It intentionally includes an interactive Q-Learning Gridworld demo to showcase learning dynamics and policy visualization.

Files added:

- `index.html` — main static site
- `styles.css` — styling
- `rl-demo.js` — interactive Q-learning demo (client-side JS)

How to preview

1. Clone the repo and checkout the `portfolio-site` branch (or open GitHub Pages after merging):

   git clone https://github.com/AjayBandiwaddar/portfolio.git
   cd portfolio
   git checkout portfolio-site

2. Open `index.html` in your browser.
